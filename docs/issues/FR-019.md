# FR-019: Meloschool Integration

Status: Future  
Labels: integration, future

## Requirement

Meloschool integration lets Meloschool admissions open EduClearance checks directly.

## User Story

As a Meloschool user, I want clearance checks inside admissions, so that the products work together later.

## Proposed Solution

Later connect Meloschool admissions to EduClearance.

## Implementation Flow

API/auth handoff and embedded clearance action.

## Technical Approach

Follow the stack and data model in `docs/Project_Requirements.md` and the rules in `docs/Coding_Guidelines.md`. Use server-side validation, school-scoped permissions, audit logging where sensitive, and transactional database operations for wallet/payment work.

## Key Considerations

Keep the MVP focused on clearance requests, not full school management. Use safe product language, minimize child/parent data, and avoid presenting missing records as clearance.
